Description

This project is a standalone Java (Processing) application designed to assist students and teachers with transcription, translation, and protein feature analysis from DNA or RNA sequences. The user can also simulate biological mutations such as insertions, deletions, and point mutations, and visualize before/after results.

Instructions

  • Run Protein_Sequence_Analyzer.exe or open the sketch in the Processing IDE
  • Enter a DNA, RNA, or protein sequence to analyze
  • Click “Apply Mutation” to simulate biological mutations
  • Compare original and mutated sequences along with protein feature changes
